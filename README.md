# Personal Expense Visualization Project

This project analyzes personal expenses and visualizes spending patterns using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains personal expense records including:
- Date
- Category
- Description
- Amount
- Payment Method

## Visualizations
This project includes:
- Category Spending Bar Chart
- Expense Distribution Pie Chart
- Daily Expense Trend Line Graph
- Payment Method Analysis

## Purpose
The goal of this project is to understand spending patterns and help manage personal finances using data visualization.
